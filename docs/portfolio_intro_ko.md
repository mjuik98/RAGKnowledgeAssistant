# 포트폴리오 소개글 모음

## 1문장 소개
공공 문서와 공개 웹문서를 적재해 질문에 대해 citation이 붙은 grounded answer를 제공하는 한국어 RAG 지식도우미를 설계·구현했습니다.

## 3문장 소개
공공 문서는 형식이 제각각이라 원하는 정보를 빠르게 찾기 어렵고, LLM 단독 응답은 근거 없는 hallucination 위험이 있습니다. 이를 해결하기 위해 PDF/TXT/DOCX와 공개 웹문서를 적재하고, 하이브리드 검색과 reranking을 거쳐 citation이 포함된 grounded answer를 반환하는 한국어 RAG 지식도우미를 구현했습니다. 또한 guardrail, no-answer, 평가 리포트, 운영 지표까지 포함해 데모를 넘어 제출형 포트폴리오 프로젝트로 정리했습니다.

## README / 노션용 상세 소개
이 프로젝트는 공공 문서 기반 한국어 RAG 시스템입니다. 문서와 공개 웹문서를 파싱하고 청크 단위로 저장한 뒤, BM25와 dense retrieval을 결합한 하이브리드 검색으로 관련 문맥을 찾습니다. 이후 heuristic reranking과 guard 정책을 거쳐 문서에 근거한 응답만 반환하고 citation을 함께 제공합니다. 질의 로그, 피드백, 운영 현황 API, JSONL 평가셋 실행, HTML 리포트 생성까지 포함해 시스템 설계와 운영 관점을 함께 보여줄 수 있도록 구성했습니다.

## 발표 시작 멘트
저는 LLM을 단순히 호출하는 데서 끝나는 프로젝트보다, 실제 문서를 다루고 검색 품질을 평가하며 운영까지 고려한 프로젝트를 만들고 싶었습니다. 그래서 공공 문서와 공개 정책 페이지를 기반으로, citation이 포함된 grounded answer를 반환하는 한국어 RAG 지식도우미를 구현했습니다.

## 면접용 30초 설명
이 프로젝트는 공공 문서를 대상으로 한 한국어 RAG 시스템입니다. 문서 적재, 하이브리드 검색, citation 기반 응답, no-answer와 guardrail, 평가 리포트와 운영 지표까지 직접 구현해서 LLM 응용과 백엔드, 평가 역량을 함께 보여주는 포트폴리오로 정리했습니다.
